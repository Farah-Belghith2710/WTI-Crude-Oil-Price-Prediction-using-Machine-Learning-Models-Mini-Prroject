# WTI Crude Oil Price Prediction

A machine learning project that predicts **West Texas Intermediate (WTI) Crude Oil prices** using historical financial market data (DJIA, NASDAQ, S&P 500, Bitcoin, Gold, Silver), time-series feature engineering, and multiple regression models — packaged with an interactive **Streamlit dashboard**.

## Overview

The pipeline:
1. Loads and cleans historical daily financial/energy market data.
2. Performs exploratory data analysis (trend plots, correlation heatmap).
3. Engineers time-series features (lags, rolling statistics, price change).
4. Selects the most relevant features via correlation filtering.
5. Trains and compares five regression models to predict WTI price.
6. Visualizes performance, residuals, and actual-vs-predicted prices.
7. Ships an interactive Streamlit app for exploration, manual prediction, model comparison, and 7-day forecasting.

## Data

The dataset (`dataase.csv`) contains historical daily values for:
- **Target**: Crude Oil (WTI) price
- **Features**: DJIA, NASDAQ, S&P 500, Bitcoin, Gold, Silver

Preprocessing steps:
- Parse `date` column and set it as the time-series index
- Convert comma-formatted numeric strings to floats
- Forward-fill then back-fill missing values

## Feature Engineering

Built on top of the cleaned WTI price series:

| Feature Type | Description |
|---|---|
| **Lag features** | `wti_lag_1`, `wti_lag_3`, `wti_lag_5`, `wti_lag_7` — past prices, capturing short-term momentum and weekly trends |
| **Rolling statistics** | 7/14/30-day rolling mean and standard deviation — capture trend smoothing and volatility |
| **Price change** | Day-over-day percentage change |

Features are further filtered to keep only those with an absolute correlation > 0.5 with the WTI target, reducing overfitting risk.

## Modeling

Data is split chronologically (80% train / 20% test, no shuffling, to respect time order) and scaled with `StandardScaler`. Five regression models are trained and compared:

- Linear Regression
- Random Forest Regressor
- XGBoost Regressor
- Gradient Boosting Regressor
- Support Vector Regressor (SVR)

### Evaluation Metrics
- RMSE (Root Mean Squared Error)
- MAE (Mean Absolute Error)
- MAPE (Mean Absolute Percentage Error)
- R² Score

The notebook ranks all models by RMSE and highlights the best performer, along with:
- Bar chart comparisons of RMSE / MAE / MAPE / R²
- Actual vs. predicted price plots for every model
- Residual analysis (residual plot + distribution) for the best model
- Full-period (train + test) actual vs. predicted visualization

## Streamlit Dashboard

An interactive app (embedded in the final notebook cell, intended to be extracted into a standalone `.py` file) provides:

- **Dataset Insights** — data preview, descriptive statistics, and target variable explanation
- **Interactive Manual Prediction** — sliders for top features to get instant WTI price predictions
- **Model Performance & Ranking** — sortable comparison table with highlighted best scores per metric, plus explanations of tree-based vs. other models
- **Prediction Visualization** — actual vs. predicted price chart, selectable by model
- **Forecast Next 7 Days** — iterative multi-step forecast using the best model, with CSV export

## Notes

- The train/test split is **chronological** (not random), which is appropriate for time-series data but means results are sensitive to which time period falls into the test set.
- The Streamlit app retrains all models on load (cached via `@st.cache_resource`) rather than loading pre-trained `.pkl` files, so the notebook's saved model artifacts and the app's models are trained independently.
- The 7-day forecast page uses a naive strategy: exogenous features (DJIA, gold, etc.) are held constant at their last known value while the WTI price is forecast iteratively.
